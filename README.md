# User Service

A Spring Boot based User Management Microservice for an E-Commerce Platform.

This service is responsible for:
- User registration
- User retrieval
- User updates
- User deletion
- User persistence using PostgreSQL
- REST API exposure

---

# Tech Stack

## Backend
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven

## Database
- PostgreSQL

## Tools
- IntelliJ IDEA
- Git & GitHub
- Postman
